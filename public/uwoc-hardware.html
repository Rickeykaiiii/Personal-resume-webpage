Underwater Wireless Optical Communication

水下無線光通信硬件

自主設計的水下無線光通信電路系統

Self-designed Hardware System for Underwater Wireless Optical Communication

項目簡介

這個項目是我在實驗室期間自主設計和搭建的水下無線光通信硬件系統。整個系統實現了從視頻採集、光信號發射、水下傳輸到接收顯示的完整鏈路。

為了方便後續的學弟學妹使用這套系統，我編寫了一份詳細的《LED 平台使用方法及注意事項》文檔，記錄了每個硬件模塊的連接方式、工作原理和常見問題排查方法。

系統包含發射端和接收端兩部分，發射端將攝像頭採集的視頻信號轉換為光信號發射，接收端通過 PIN 陣列接收光信號並還原為視頻顯示。

系統架構

整個通信系統由發射端和接收端兩條鏈路組成：

發射端鏈路

攝像頭 → FPGA 發射端 → 信號放大模塊 → 功率放大模塊 → LED 發射端

接收端鏈路

PIN 陣列接收端 → FPGA 接收端 → HDMI 顯示屏

硬件模塊

信號放大模塊

將 FPGA 輸出的 0-2.5V 電平抬高，為後續電路提供足夠的驅動能力。需要 ±5V 雙電源供電，正常工作時芯片溫度微溫。

功率放大模塊

進行方波整形和功率放大，將 3.3V 信號抬高到接近電源電壓，直接驅動 LED 發光。芯片工作時溫度較高，需注意散熱。

LED 陣列發射端

實驗室有兩種型號：大重量型（9-12V 供電，可短距離傳輸）和小重量型（7-8V 供電，長距離傳輸）。LED 發光穩定，無頻閃。

TIA 光電二極管電路

跨阻放大器（TIA）電路負責將 PIN 光電二極管接收到的微弱光電流轉換為電壓信號。採用高增益、低噪聲設計，確保信號的靈敏度和信噪比。

接收端後級放大電路

對 TIA 輸出的電壓信號進行進一步放大和整形，提升信號幅度至 FPGA 可識別的電平範圍，同時進行濾波處理以降低噪聲干擾。

FPGA 發射端黑盒

將攝像頭的 HDMI 視頻信號轉換為電平信號。通過 SMA 連接線輸出到後續放大電路。正常工作時會有綠光紅光指示，風扇運轉。

FPGA 接收端黑盒

接收來自後級放大電路的信號，進行解碼處理後通過 HDMI 輸出到顯示屏。與發射端配合完成完整的視頻傳輸鏈路。

技術細節

信號處理

FPGA 負責視頻信號的編解碼和電平轉換，實現高速數據傳輸。

多級放大

通過信號放大和功率放大兩級電路，確保 LED 有足夠的驅動電流。

電源設計

不同模塊需要不同電源配置（±5V、單電源），需要仔細連接避免反接。

連接方式

模塊間使用 SMA 連接線傳輸射頻信號，確保信號完整性。
