# 蜀夢千載：三星堆文物線上 3D 手勢互動展
**(Shu Dream A Millennium: Interactive 3D Sanxingdui Exhibition with Gesture Control)**

## 專案簡介 (Project Overview)

「蜀夢千載」是一個融合**文化遺產**與**前沿人機互動技術**的 Web3D 展覽專案。本專案透過高精度的 3D 建模技術還原神秘的古蜀三星堆文物（如青銅大立人、青銅神樹、黃金面具等），並結合先進的電腦視覺技術，讓使用者**只需透過電腦視訊鏡頭揮動雙手**，即可在瀏覽器內零距離、全方位地賞析千年國寶。

本專案旨在打破實體博物館的地理限制，以最自然、直覺的「凌空手勢」互動，為使用者帶來沉浸式的數位考古體驗。

## 核心功能 (Core Features)

* **🤲 無接觸手勢操控 (Markerless Gesture Control)：** 透過 AI 視覺演算法捕捉手部骨架節點，實現無延遲的凌空互動。無需穿戴任何設備，揮手即可與文物對話。
* **🏺 高精度 3D 文物還原 (High-Fidelity 3D Rendering)：** 採用 PBR (Physically Based Rendering) 材質技術，完美重現青銅器的鏽跡、黃金的光澤與玉石的溫潤。
* **🌐 零安裝網頁體驗 (Web-Native Experience)：** 基於 WebGL 構建，隨點即看。支援跨平台（PC/平板），無需下載任何應用程式或外掛程式。
* **🎧 沉浸式文化解說 (Immersive Storytelling)：** 隨著使用者旋轉文物到特定角度，系統會自動觸發該部位的圖文與語音解說，揭開古蜀文明的神秘面紗。

## 技術架構 (Technology Stack)

本專案採用純前端架構與端側 AI 推理，確保使用者的互動隱私與極致流暢度：

| 領域 | 使用技術 / 框架 | 說明 |
| :--- | :--- | :--- |
| **前端框架** | React / Vue.js | 處理 UI 介面、展廳導航與狀態管理。 |
| **3D 引擎** | Three.js (WebGL) | 負責 3D 模型的加載 (GLTF/GLB)、光影渲染與場景構建。 |
| **手勢辨識** | MediaPipe (TensorFlow.js) | 在瀏覽器端即時進行 21 個手部關鍵點（Hand Landmarks）的追蹤與姿態預測。 |
| **模型處理** | Blender | 3D 模型的前期拓撲最佳化、烘焙與材質貼圖處理。 |

## 互動指南 (Interaction Guide)

為了提供最直覺的體驗，我們設計了符合人類直覺的手勢指令：

1.  **單手握拳並拖曳 (Grab & Drag)：** 旋轉 3D 文物，360 度無死角觀賞。
2.  **雙手捏合 / 張開 (Pinch / Spread)：** 縮放文物模型，放大檢視青銅器上的饕餮紋理。
3.  **食指懸停 (Index Point & Hover)：** 瞄準文物上的「知識熱點 (Hotspot)」，停留 2 秒即可展開詳細解說面板。
4.  **手掌平攤並向左/右揮動 (Swipe Left/Right)：** 切換至上一件或下一件展品。

## 重點展品清單 (Featured Artifacts)

* **青銅縱目面具：** 體驗古蜀人對「千里眼」與「順風耳」的超自然崇拜。
* **青銅神樹：** 探索古代中國宇宙觀與太陽崇拜的具象化體現。
* **金面銅人頭像：** 感受青銅與黃金結合的威嚴與神秘。
* **玉璋：** 了解古蜀國的祭祀禮儀與玉器加工工藝。

## 專案意義 (Impact)

1.  **數位典藏活化：** 讓靜態的文物「活」起來，拉近大眾與歷史的距離。
2.  **無障礙教育：** 提供了有別於滑鼠和鍵盤的互動方式，為不同年齡層（尤其是學童）帶來寓教於樂的歷史學習體驗。
3.  **技術探索：** 展示了 Web 端 AI 結合 3D 渲染在元宇宙 (Metaverse) 與數位展陳領域的巨大潛力。
