水下YOLO檢測海洋垃圾

基於 TensorFlow Lite 的水下垃圾檢測與清理視覺系統

Underwater Marine Debris Detection and Cleaning Vision System

項目簡介

這個項目聚焦於水下海洋垃圾的視覺檢測與後續清理流程，目標是在複雜水下環境中識別常見塑料垃圾，為自主水下機器人提供感知能力。系統以輕量化推理為核心，採用 TensorFlow Lite 模型完成目標檢測，再將檢測結果作為後續距離估計與機械臂抓取的輸入。

模型基於約 1500 張圖像進行訓練，現階段可識別吸管、瓶子、罐子、袋子和杯子 5 類垃圾，在專案 README 記錄中整體準確率約為 80%。相較於完整尺寸模型，TFLite 更適合部署在資源受限的邊緣設備或機器人平台上。

系統流程

1. 水下垃圾目標檢測

使用 TensorFlow Lite 檢測模型對圖片或視頻幀進行推理，輸出垃圾類別、邊界框位置與置信度，完成塑料垃圾的初步定位。

2. 距離估計

結合 OpenCV 與三角測量方法估算目標距離，讓系統不只知道垃圾在哪裡，也能提供抓取和接近所需的空間信息。

3. 機械臂清理與收納

在完整設計中，檢測到的垃圾會由機械臂執行收集，並進一步壓縮後存入網袋，形成從感知到清理的閉環構想。

項目以 Python 為主體，結合輕量級深度學習與傳統視覺算法，兼顧檢測效果與部署可行性：

Python 3.9+

負責模型調用、圖像處理、腳本編排與整體流程控制。

推理引擎

TensorFlow Lite

採用輕量模型進行垃圾分類與檢測，適合邊緣側部署與快速推理。

視覺處理

項目結構與使用方式

專案結構分為檢測模組、距離計算模組、TFLite 模型文件、圖片輸入與輸出結果資料夾。實際使用時，可將待測圖片放入指定資料夾，再透過批次腳本或手動命令執行檢測。

→ `Detection/`：圖片與視頻檢測腳本

→ `Distance Calculation/`：立體視覺距離計算

→ `Tflite Model/`：模型與標籤文件

→ `my_images/`：輸入圖片資料夾

→ `output/`：檢測後輸出結果

技術亮點

→
輕量化部署： 選用 TensorFlow Lite 作為推理引擎，降低模型部署成本，更適合嵌入式或水下機器人平台。
→
檢測與測距結合： 不僅辨識垃圾類別，同時保留接入距離估計模組的能力，讓後續抓取控制更可行。
→
面向清理任務設計： 項目不是單純的視覺識別 Demo，而是面向「發現垃圾、接近垃圾、抓取垃圾」完整任務鏈設計。

後續優化方向

→ 擴展更多垃圾類別，提高模型泛化能力

→ 優化準確率與推理速度，提升實際部署表現

→ 從圖片檢測延伸到實時視頻流檢測

→ 將檢測結果更完整地接入機械臂控制與清理流程
