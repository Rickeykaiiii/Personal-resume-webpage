簡歷生成器

基於 JSON 格式的在線簡歷生成工具

JSON-based Online Resume Builder with Real-time Preview

🎯 體驗線上演示

項目簡介

這是一個基於 React 的在線簡歷生成工具，部署在 GitHub Pages 上。採用「左側編輯、右側實時預覽」的設計，讓用戶可以即時看到簡歷的最終效果。

支持基本信息、教育背景、實習經歷、項目經驗、技能和語言等完整的簡歷模塊。用戶可以自由切換模塊顯示、強制分頁，還支持 HTML 格式編輯，靈活度很高。

核心功能

實時預覽編輯

左側編輯器修改內容，右側立即顯示最終效果，所見即所得的編輯體驗。

JSON 數據管理

支持導入/導出 JSON 格式數據，方便保存和遷移簡歷內容，數據存儲在瀏覽器本地。

頭像管理

支持本地上傳頭像或使用 URL 鏈接，靈活管理個人照片。

PDF 導出

使用瀏覽器原生打印功能導出 PDF，無需額外依賴，支持 A4 頁面佈局控制。

二維碼生成

自動生成個人網站二維碼，方便在簡歷上展示在線作品集。

技術實現

項目採用現代前端技術棧，純前端實現，無需後端服務：

前端框架

React 19.2.0

使用最新版本 React，配合 Vite 構建工具實現快速開發。

樣式方案

Tailwind CSS v4.1.18

使用 Tailwind CSS 實現響應式設計，配合 PostCSS 和 Autoprefixer。

圖標庫

Lucide React

使用 Lucide 圖標庫提供清晰美觀的界面圖標。

字體

Noto Sans SC

通過 Google Fonts 引入思源黑體，支持中文顯示。

架構設計

→
單文件架構： 核心邏輯集中在 App.jsx 文件中，代碼結構清晰，易於維護和理解。
→
純前端應用： 所有數據存儲在瀏覽器本地，無需後端服務器，部署簡單，隱私安全。
→
響應式設計： 針對現代瀏覽器（Chrome、Firefox、Safari）優化，支持不同屏幕尺寸。

使用體驗

模塊化編輯

可以自由切換各個簡歷模塊的顯示狀態，根據需要定制簡歷內容。

強制分頁

支持在特定位置強制分頁，精確控制 A4 頁面佈局。

HTML 編輯

支持 HTML 格式編輯，可以添加鏈接、加粗等富文本效果。

A4 頁面輔助線

提供 A4 頁面輔助線，幫助用戶更好地控制內容佈局。

部署方式

項目部署在 GitHub Pages 上，訪問地址：
